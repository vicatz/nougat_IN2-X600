# nougat_IN2-X600
<h1>Cara Install :<h1>

1) masuk twrp
2) masuk ke menu wipe
3) pilih advance wipe
4) centang:
5) wipe cache.
6) wipe dalvik cache
7) wipe data	
8) wipe system
9) flash rom
10) flash gapps
11) flash patch B1/D1 (user Note 2 pro A1 lewatin)
12) flash patch fix xxxx (jika ada)
13) jika waktu install rom lwt twrp ketemu error "installation error-aborted xxxx" lanjut baca dibawah
