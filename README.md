# nougat_IN2-X600
<h1>Installation :<h1>

  1) wipe cache.
  2) wipe dalvik cache
  3) wipe data	
  4) wipe system
  5) flash rom
  6) flash gapps
  7) flash patch B1/D1 or H532 (user Note 2 pro A1 skip this step)
  8) flash fix xxxx (if any)
  9) if u facing errors during installation please see FAQ
